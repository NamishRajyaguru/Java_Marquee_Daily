# Java Marquee Daily

A structured and progressive repository dedicated to daily Java practice, ranging from fundamental concepts to advanced logical problem-solving.

## Overview

This workspace serves as a continuous learning environment for mastering Java. It's organized into dedicated modules to ensure a systematic approach to coding:

- **Basics**: Core Java fundamentals, syntax, and foundational exercises.
- **Functions**: Method definitions, recursion, scopes, and memory management.
- **Math and Logic**: Number manipulation, logic-based puzzles, and algorithmic thinking.
- **Patterns**: Nested loops and complex geometric output challenges.
- **Arrays**: Data structures, searching algorithms, and element manipulation.

## Structure

Each folder contains comprehensive study notes (`notes.txt`) alongside sequentially numbered code exercises (`Test00.java`, `Test01.java`, etc.) that directly implement the concepts covered.

*Designed to build strong, consistent problem-solving skills in Java.*
